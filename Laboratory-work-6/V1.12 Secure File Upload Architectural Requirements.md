## CWE-646: Залежність від імені файлу або розширення файлу, наданого ззовні (Reliance on File Name or Extension of Externally-Supplied File)
Програмне забезпечення дозволяє завантажувати файл, але воно покладається на ім'я файлу або розширення файлу для визначення відповідної поведінки. Це може бути використано зловмисниками для того, щоб файл був неправильно класифікований і оброблений небезпечним чином.
